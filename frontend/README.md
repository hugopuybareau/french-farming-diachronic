# France Agri Map

Carte interactive des données du Recensement Agricole 2020.

## Technologies

- React + TypeScript
- Vite
- Tailwind CSS + shadcn/ui
- D3.js pour la visualisation cartographique
- Zustand pour la gestion d'état

## Installation

```sh
npm install
npm run dev
```

## Scripts

- `npm run dev` - Serveur de développement
- `npm run build` - Build de production
- `npm run lint` - Vérification ESLint
- `npm run test` - Tests unitaires
